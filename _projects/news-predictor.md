A deep learning–based news classification application that automatically analyzes and categorizes news articles into predefined thematic topics. It uses neural network models to perform both single-article inference and batch processing, enabling efficient handling of individual inputs as well as large datasets. The application also includes a feature to retrieve and organize the latest news by category, allowing users to stay updated on current events across different themes. This functionality is supported through integration with a free external news API, which provides real-time news data but is subject to usage limits and rate restrictions.

[Try the app](https://newschat-laurapa.streamlit.app/)
